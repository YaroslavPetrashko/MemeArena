# Card art

Drop card images here named by slug, e.g. `pepe-the-frog.png`, `dogwifhat.png`.
Recommended: 512×512 or 4:3, PNG/WEBP. Until present, the app renders a polished
gradient + initials placeholder (no broken images). No code changes needed.

Expected filenames:
pepe-the-frog, dogwifhat, mog-cat, popcat, peanut-the-squirrel, moo-deng-hippo,
tung-tung-tung-sahur, gigachad, ballerina-cappuccino, tralalero-tralala,
bombardino-crocodilo, wojak, sigma-cat, cappuccino-assassin
